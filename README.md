# Home Board 🏠📋

Um sistema simples e intuitivo para gerenciamento de tarefas domésticas, permitindo organizar atividades da casa em diferentes etapas de progresso.

## ✨ Objetivo

O **Home Board** foi criado para ajudar no controle das tarefas do dia a dia de forma prática e visual.  
Com ele, é possível adicionar atividades, acompanhar o andamento e marcar tarefas como concluídas.

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas
- 🛠️ Mover tarefas para “Em Andamento”
- ✅ Finalizar tarefas
- ⛔ Remover tarefas
- 📊 Contador automático de tarefas por categoria
- 🎨 Interface moderna e responsiva

---

## 🖼️ Estrutura do Sistema

O sistema é dividido em 3 colunas principais:

| Status | Descrição |
|---|---|
| 📍 Para Fazer | Tarefas que ainda não começaram |
| 🛠️ Em Andamento | Tarefas que estão sendo realizadas |
| ✅ Concluído | Tarefas finalizadas |
